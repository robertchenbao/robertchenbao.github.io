Projects

  • Using Dominating Sets to Block Contagions in Social Networks: A research paper about blocking contagions (like virus or misinformation) in a population. Accepted to ASONAM 2022
  • GlucOverwatch: A Python program that monitors glucose levels of Type-1 diabetes patients. Winner of UVA's Hack4Health 2022
  • The Pedometer Algorithm: A Python program for counting steps using just smartphones sensor data, with over 90% accuracy
  • The Search Toolkit: A React web app that makes building complex Google query string easier
  • Code Warehouse: A web app that helps engineers scientists manage and share code snippets
  • Lucid Allergy Forecast: A Python web app that visualizes local 5-day allergen level forecast data
  • Twitter Analytic Program: A machine-learning web app that can analyze and visualize over 20,000 tweets. Winner of Fort Myer's Thomas A. Edison Inventor Fair in 2020
  • Marketing from Media: A data science project on Twitter. I mined a 3.5 GB tweet dataset to analyze public perception on Tesla, and its connection to Tesla car sales/stock price. Winner of Fort Myer's Thomas A. Edison Science Fair in 2019 and 2020
